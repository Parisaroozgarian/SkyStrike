System Overview

The Aviation Threat Simulator is a Master's-level cybersecurity training platform designed to simulate sophisticated cyber attacks against aircraft systems.

Key Features
  • Multi-Aircraft Fleet Management: Monitor and attack multiple aircraft simultaneously
  • Realistic Attack Vectors: GPS spoofing, control surface manipulation, communication jamming
  • Advanced Defense Systems: Multi-layered protection with automated response
  • Professional Analytics: Comprehensive threat analysis and reporting
  • Enterprise Architecture: Scalable, secure, and maintainable codebase
System Components
ADS-B System

Automatic Dependent Surveillance-Broadcast transponder simulation

Flight Control

Primary flight control surfaces and autopilot systems

Communications

VHF radio and data link communication systems

System Architecture

Professional Architecture Pattern

The application follows enterprise-grade architectural principles:

  • Application Factory Pattern: Modular Flask application initialization
  • Service Layer Architecture: Separated business logic in dedicated services
  • Data Model Abstraction: Professional data classes with validation
  • Configuration Management: Environment-based configuration system
  • Structured Logging: JSON-based logging with security audit trails
  • Input Validation: Comprehensive request validation and sanitization

Attack Vectors

ADS-B Attacks
  • GPS Spoofing: False position data
  • Altitude Spoofing: Dangerous altitude reports
  • Man-in-the-Middle: Position data interception
  • Replay Attacks: Historical data replay
Flight Control Attacks
  • Input Jamming: Extreme control positions
  • Control Freeze: Surface position lock
  • Bus Interception: Command modification
  • Command Replay: Previous maneuvers
Communication Attacks
  • Message Injection: False emergency signals
  • Radio Jamming: Communication blackout
  • Channel Interception: ATC impersonation
  • Message Replay: Outdated instructions

Defense Systems

Multi-Layered Defense Framework
Intrusion Detection

Monitors for unauthorized access and suspicious activities across all aircraft systems.

Data Validation

Verifies integrity of incoming data from aircraft systems and external sources.

Encryption System

Protects communications with automatic key rotation capabilities.

Backup Systems

Redundant systems that activate when primary systems are compromised.

Automated Response Protocols
  • Immediate threat detection and alerting
  • Automatic system isolation for compromised aircraft
  • Emergency backup system activation
  • Real-time threat correlation and analysis
  • Automated recovery procedures
